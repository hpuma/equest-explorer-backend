Equest Explorer



Equest explorer is a web application built using React, Axios, Vite, and Ant Design (antd) that provides users with a comprehensive toolset for tracking stocks, accessing news related to financial markets, and analyzing stock data.


Features

  • Stock Quote for Ticker: Users can input a stock ticker symbol to retrieve real-time stock quotes.
  • Enable/Disable Stock Chart via Trading View: Users can toggle the display of a stock chart powered by TradingView to visualize stock data.
  • Theme Changes (Dark vs Light): Users can switch between dark and light themes for better readability and user preference.
  • News Sources Ingested from Multiple APIs: News articles related to stocks are sourced from multiple APIs including News API, Alpha Vantage, Marketaux, Bing, NewsData, and TheNewsAPI.
  • Ticker Search (On Key Type): As users type, the application dynamically searches for stock tickers and provides suggestions for auto-completion.
  • News Preview Viewer with Vertical Adjustment: Users can view a preview of news articles with the ability to vertically adjust the content for better readability.
  • Responsive Table with Filtering and Sorting: A responsive table displays stock data with features for filtering and sorting based on various parameters such as stock symbol, price, volume, etc.

Integrations

  • NewsAPI
  • AlphaVantage
  • Marketaux
  • Bing (2/7/2024)
  • NewsData (2/8/2024)
