Description

Equest Explorer backend provides news articles related to stocks via News REST API integrations such as

  • NewsAPI
  • AlphaVantage
  • Marketaux
  • Bing (2/7/2024)
  • NewsData (2/8/2024)
  • GNews (2/8/2024)
  • TheNewsAPI (2/8/2024)

Prerequisites

.env file in root directory
